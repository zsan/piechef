# HOW TO RUN

- npm install
- run redis-server
- ./bin/devserver
- go to http://127.0.0.1:3000/login (user: piechef, password: 12345)
- add some pies
- goto http://127.0.0.1:3000/admin/menu/stage (click ready, oven or gone and watch with other browser window and see the changes)
